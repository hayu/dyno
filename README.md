# Learning html with javascript and css, and their equivalents in coffeescript

## To get up and running

```bash
$ brew install npm
$ brew install coffeescript
$ npm install --save-dev coffeescript
$ cd hello/coffee
$ coffee -b -c -o dist hello.coffee

```
